## :computer: Run Locally

Clone the project

```bash
  git clone git@github.com:clair-daisies/Avocado-Full-Ecommerce-Store.git
```

Go to the project directory

```bash
  cd Avocado-Full-Ecommerce-website
```

Spin up the server

```bash
  serve via Live-server
```
